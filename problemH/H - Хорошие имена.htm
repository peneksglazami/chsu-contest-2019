Входной файл: input.txt

Выходной файл: стандартный поток вывода («вывод на экран»)

Ограничение по времени: 1 секунда на тест

Ограничение по памяти: 32 Мб

Условие задачи

На раскопках древнего поселения, находящегося на территории современной Эйсляндии, нашли камень с высеченными на нём буквами, который по мнению археологов использовался прародителями эйсляндцев для составления имён, приносящих успех и удачу их владельцам.
Камень имеет форму квадрата, который разбит на клетки. В каждой клетке высечена одна буква. По поверьям древних имя считалось приносящим успех и удачу в том случае, когда его можно было составить из букв, высеченных в клетках, выполняя последовательно переход от одной клетки к другой. Переходить из одной клетки в другую можно по одному из четырёх направлений: вверх, вниз, влево, вправо. При этом возвращаться в уже пройденную клетку нельзя.
Вам предстоит разработать программное обеспечение, которое определяет, можно ли указанное имя найти на камне или нет.

Входные данные

Первая строка файла input.txt содержит имя, длина которого не превышает 10 букв.
Вторая строка содержит число N (1 <= N <= 10), определяющее количество строк и столбцов в клеточной таблице камня. Далее следует N строк, каждая из которых содержит N букв, соответствующих одной строке клеточной таблицы камня.

Выходные данные

В стандартный поток необходимо вывести слово YES, если имя можно найти на камне, или NO в противном случае.

input.txt

Стандартный поток вывода

JAVA

5

AJBVE

VAGGH

TYTRJ

YES
