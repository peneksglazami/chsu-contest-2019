Входной файл: input.txt

Выходной файл: стандартный поток вывода («вывод на экран»)

Ограничение по времени: 1 секунда на тест

Ограничение по памяти: 32 Мб

Условие задачи

Мэр Эйсбурга очень любит изюм, поэтому на окраине города разбил виноградник. Для того чтобы кусты винограда были защищены он назойливых глаз журналистов, он решил свой сад окружить сплошным забором, при этом хочет это сделать так, чтобы расстояние он любого куста винограда до забора не было меньше 1 метра, а общая длина забора была минимальной.
Мэр Эйсбурга обратился к вам с просьбой помочь ему определить минимальную длину забора.

Входные данные

Первая строка файла input.txt содержит имя, длина которого не превышает 10 букв.
Вторая строка содержит число N (1 <= N <= 100) - количество кустов винограда. Далее следует N строк, каждая из которых содержит два целых числа X и Y (0 <= X, Y <= 100) задающих координаты куста винограда.

Выходные данные

В стандартный поток необходимо вывести минимальную длину забора с точностью до двух знаков (используйте математическое округление). В качестве разделителя целой и дробной частей используйте точку.

input.txt

Стандартный поток вывода

4

0 0

5 0

0 5

5 5

26.28
