Входной файл: input.txt

Выходной файл: стандартный поток вывода («вывод на экран»)

Ограничение по времени: 1 секунда на тест

Ограничение по памяти: 32 Мб

Условие задачи

Территорию Эйсляндии населяет множество народов, различных по культуре и внешнему облику, но говорящих на одном языке. Каждый народ предпочитает носить одежду определённого цвета, который отличается от цвета одежды других народов. Народы имеют разные традиции, порой традиции одних народов противоречат традициям других народов. Поэтому жители каждого города следуют традициям того народа, представителей которого проживает в этом городе больше всего. Если оказывается, что таких народов несколько, все жители города следуют традициям самого миролюбивого народа с белым цветом одежды (белый цвет обозначается нулём).

Путешественник стоит на высоком холме недалеко от входа в город. С этого холма он видит цвет одежды каждого жителя города. Путешественник торопится войти в город, ему важно быстро определить, традициям какого народа следовать в этом городе.

Входные данные

Входной файл input.txt состоит из двух строк. Первая строка входного файла содержит целое число N - количество жителей города, которых видит путешественник (1 ≤ N ≤ 104). Вторая строка содержит N натуральных чисел, разделенных пробелами. Каждое число сi – это цвет одежды i-го жителя (1 ≤ ci ≤ 100).

Выходные данные

В стандартный поток выводится единственное целое число – цвет одежды народа, традициям которого следуют жители города.
