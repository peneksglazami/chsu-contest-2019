Входной файл: input.txt

Выходной файл: стандартный поток вывода («вывод на экран»)

Ограничение по времени: 3 секунды на тест

Ограничение по памяти: 32 Мб

Условие задачи

В Эйсляндии решили провести чемпионат страны по покеру. На участие в чемпионате прислали заявки команды из нескольких городов. Турнир будет проводиться в колонном зале дворца президента Эйсляндии, в котором будут располагаться игральные столы. Каждый стол имеет ограничение по тому, сколько человек может за ним сидеть. Кроме этого для обеспечения честности игры в регламент чемпионата было добавлено правило, запрещающее нахождение за игровым столом двум и более игрокам, представляющий один и тот же город.
Вам поручено до начала чемпионата определить, смогут ли все заявленные игроки быть рассажены за игровыми столами в соответствии с регламентом.

Входные данные

Первая строка файла input.txt содержит два числа N (1 <= N <= 100) и M (1 <= M <= 100), определяющих соответственно количество городов, приславших заявку, и количество столов.
Вторая строка файла содержит N чисел K1, K2, …, Kn (1 <= K1, K2, …, Kn <= 100), где Ki - количество человек в команде i-го города.
Третья строка файла содержит M чисел S1, S2, …, Sm (1 <= S1, S2, …, Sm <= 100), где Si - количество мест за i-ым игровым столом.

Выходные данные

В стандартный поток необходимо вывести слово YES, если рассадить игроков за столами получится, или NO в противном случае.

input.txt

Стандартный поток вывода

3 3

1 2 3

2 2 2

YES

4 3

1 2 3 4

1 2 4

NO
