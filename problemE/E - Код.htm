Входной файл: input.txt

Выходной файл: стандартный поток вывода («вывод на экран»)

Ограничение по времени: 1 секунда на тест

Ограничение по памяти: 32 Мб

Условие задачи

Мэр Эйсбурга ведет обширную переписку с коллегами из окрестных регионов. Часть этой переписки является государственной тайной.  И хотя сообщения передаются по защищенному каналу связи, было принято решение кодировать их специальным неравномерным префиксным кодом. Это означает, что каждому символу сообщения ставится в соответствие последовательность нулей и единиц не обязательно одинаковой длины для всех символов. Кроме того, для возможности однозначного декодирования код никакого символа не может быть началом кода другого. И наконец, после передачи всего закодированного сообщения посылается целое число K – общее количество символов исходного текста. Помогите по имеющейся таблице кодов символов и закодированному сообщению шифровальщику мэра определить число K.

Входные данные

Первая строка файла  input.txt содержит одно натуральное число N – количество различных символов сообщения (1≤N ≤ 200). Далее следует N строк, в каждой из которых последовательность нулей и единиц без пробелов – коды отдельных символов. Длина каждого кода не превышает 1000 нулей и единиц. Последняя строка входного файла содержит закодированное сообщение – непустую последовательность нулей и единиц, длиной не более 1000000.

Выходные данные

В стандартный поток выводится одно целое число K – количество символов исходного сообщения.
